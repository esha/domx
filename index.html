    • Domx

    DOM extending successor to HTML.js

    Project maintained by esha Hosted on GitHub Pages — Theme by mattgraham

    domx is a small, extensible library to help you enjoy the DOM in a simple, direct, and very powerful way.

    Please check out the demo, the API and the F.A.Q..

    Standard Version:

    Download: domx.min.js or domx.js
    Bower: bower install domx
    NPM: npm install domx
    Component: component install esha/domx

    Includes each(), document.extend(), query(), queryAll(), only(), add(), remove(), toArray(), emmet abbreviations in add(), and dot-traversal:

    • domx.base.js
    • domx.emmet.js
    • domx.elements.js
